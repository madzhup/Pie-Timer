Pie Timer
=========

Простейший [jQuery](http://jquery.com "jQuery") плагин для создания кругового таймера/индикатора, также использует [Raphael JS](http://raphaeljs.com "Raphael JS").

Процент указывается в `data-percents="45"`


```javascript
$('selector').arc({
	width: 100,
	height: 100,
	outRadius: 50,
	inRadius: 30,
	color: '#fff',
	fontSize: 18,
	easing: 'easeIn',
	time: 2000
});
```
